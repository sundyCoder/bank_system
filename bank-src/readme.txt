					简单的银行系统
本项目设计到的知识
	1 控制台的简单交互
	2 流程控制的灵活应用
	3 了解项目的几个四个层次
   a.表现层/展现层(presitent layer)：基于界面的，直接提供给用户的。console,GUI(QT,MFC), WEB UI,
   b.业务逻辑层(business layer)：操作业务数据(事务处理，出错处理，业务逻辑处理)
   c.持久层(persistent layer)：对数据库的相关操作(CRUD)，XML,JSON 
   d.数据库层(database layer)：sqlite,mysql,sqlserver,db2,oracle
